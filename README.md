# To Do app

## This project creates a To Do app that you can use by visiting the heroku Link
[To Do App Link](https://react-express-todo-app.herokuapp.com/)

## Stack
### Client Side
1. React (JSX)
2. Redux with Sagas
3. BootStrap
### Server Side
4. Express
5. Node.js
6. MongoDB
### Hosting
7. Deployed to Heroku  
### Build Tools
8. Babel
9. Webpack
